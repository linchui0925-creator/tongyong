# 文件提取功能开发计划

## 需求概述

- **目标**：用户拖拽文件（图片/PDF）到输入框 → 后端提取文字 → 发送给大模型
- **节省 token**：后端提取关键信息，不上传原文件给大模型
- **OCR 方案**：PaddleOCR（免费开源，支持中文）

---

## 技术架构

### 文件处理流程

```
前端                              后端                                大模型
  │                                 │                                    │
  ├─ 拖拽文件到输入框 ─────────────> │                                    │
  │  (显示文件名+缩略图)           │                                    │
  │                                 │                                    │
  ├─ 点击发送 ──────────────────>  │                                    │
  │  (文件+文字)                  │                                    │
  │                                 ├─ 解析文件 ──────────────────>    │
  │                                 │  (PDF→文字/图片→文字)           │
  │                                 │                                    │
  │                                 ├─ 提取关键信息 ──────────────>    │
  │                                 │  (返回纯文本)                  │
  │                                 │                                    │
  ├────────────────────────────────<─┤                                    │
  │  (大模型回复)                   │                                    │
```
